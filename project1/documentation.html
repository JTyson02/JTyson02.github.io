Design notes

Well, the design process for this wasn't too detailed. My portfolio's goal is to show off my skills in designing games, though it pulls double duty in helping show off my HTML skills. My audience is anyone who would hire me, to be completely honest. I've included pretty much everything I wanted to, except for actual project examples to implement. My organization is straightforward: flexbox and divs. I probably should have done a grid layout, but at this point it would take too much time to restructure everything. The reason I think I should have used grid is because I had trouble getting the text to properly resize, and I could have done some more interesting layouts. Still, I like the simplicity of the current design. I think it fits my personality well.
